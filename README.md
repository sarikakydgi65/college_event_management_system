🎓 College Event Management System

📌 Project Overview

The College Event Management System is a web application developed to simplify the management of college events. It enables students to view and register for events, while administrators can create, update, and manage events efficiently.

The system reduces manual work, improves event organization, and provides a centralized platform for managing college activities.

---

✨ Features

Admin

- Secure Login
- Add New Events
- Update Event Details
- Delete Events
- View Registered Students
- Manage Event Categories

Student

- Register/Login
- View Upcoming Events
- Register for Events
- View Registration Status
- Update Profile
